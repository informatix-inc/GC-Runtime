# GC Runtime
株式会社Informatixの製品「GeoConic」「GC Server」等の動作を保証するランタイムです。  
AdoptOpenJDKのソース([8u222_b10](https://github.com/AdoptOpenJDK/openjdk-jdk8u/tree/jdk8u222-b10))を取得し、修正を加えたものを公開しています。
